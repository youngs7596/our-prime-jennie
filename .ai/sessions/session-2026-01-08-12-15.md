# Session Handoff: Dashboard System Status Fix & MCP Config

## 요약 (Summary)
대시보드의 시스템 상태 페이지에서 Docker와 RabbitMQ 정보가 표시되지 않는 문제를 해결했습니다. `dashboard-backend` 코드에서 데코레이터 서명 문제와 Docker Socket 전송 계층 문제를 수정했습니다. 또한, MariaDB MCP 설정 파일(`settings.json`)이 숨겨진 폴더에 격리되어 있던 것을 확인하고, Docker 환경(Port 3307)에 맞게 설정을 업데이트했습니다.

## 변경 사항 (Changes)
- `services/dashboard/backend/routers/system.py`:
    - `functools.wraps` 적용으로 캐시 데코레이터가 FastAPI 엔드포인트 서명을 가리지 않도록 수정 (RabbitMQ 조회 422 에러 해결).
    - Docker 소켓 연결 시 `httpx.AsyncHTTPTransport` 사용으로 변경 (Docker 조회 에러 해결).
- `~/.gemini/settings.json`:
    - MariaDB 포트를 3306(Local)에서 3307(Docker)로 변경.
    - 사용자(`jennie`) 및 비밀번호 업데이트.
- `CHANGELOG.md`: 금일 작업 내역 추가.
