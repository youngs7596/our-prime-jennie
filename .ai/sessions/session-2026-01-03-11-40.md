# Session Handoff: Dashboard Settings Toggle Fix
## 요약 (Summary)
대시보드의 "운영 설정 > 장 시간 체크 비활성화" 토글이 동작하지 않는 문제를 디버깅하고 수정했습니다. `docker-compose.yml`의 환경변수 우선순위 문제와, 서비스 코드에서 `ConfigManager`를 우회하여 환경변수를 직접 읽는 문제를 해결했습니다. 또한, 프론트엔드에서 API 응답 데이터를 잘못 참조하는 버그를 수정했습니다.

## 변경 사항 (Changes)
- `docker-compose.yml`: `buy-scanner`, `price-monitor` 서비스의 `DISABLE_MARKET_OPEN_CHECK` 환경변수 제거
- `services/buy-scanner/main.py`: `os.getenv` 대신 `scanner.config.get_bool` 사용으로 변경
- `services/price-monitor/monitor.py`: `ConfigManager`를 통해 `DISABLE_MARKET_OPEN_CHECK` 설정 조회하도록 변경
- `services/dashboard/frontend/src/pages/System.tsx`: `configData.configs` 접근 오류 수정 (`configData` 배열 직접 조회)
- `CHANGELOG.md`: 작업 내용 업데이트

## 다음 단계 (Next Steps)
- 대시보드 나머지 기능들의 정상 동작 여부 확인 (스케줄러 제어 등)
- 운영 모니터링: 장외 시간(주말/야간)에 토글을 켰을 때 실제 서비스들이 동작하는지 로그 확인
