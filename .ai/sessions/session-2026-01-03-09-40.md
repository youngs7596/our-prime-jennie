# Session Handoff: Dashboard Service Control

## 요약 (Summary)

대시보드 System 페이지에서 스케줄러 작업(scout-job, news-crawler 등)을 직접 제어할 수 있는 기능을 구현했습니다. 실행/일시정지/재개 버튼과 DISABLE_MARKET_OPEN_CHECK 설정 기반이 추가되었습니다.

## 변경 사항 (Changes)

### Backend
- `services/dashboard/backend/routers/scheduler.py` (NEW): 스케줄러 서비스 프록시 API (GET/PUT/POST jobs 엔드포인트)
- `services/dashboard/backend/main.py`: scheduler 라우터 등록
- `shared/settings/registry.py`: `DISABLE_MARKET_OPEN_CHECK` 운영 설정 추가 (Operations 카테고리)

### Frontend
- `services/dashboard/frontend/src/lib/api.ts`: `schedulerApi` 추가 (getJobs, runJob, pauseJob, resumeJob, updateJob)
- `services/dashboard/frontend/src/pages/System.tsx`: 
  - Scheduler Jobs 섹션 확장 (실행/정지/재개 버튼)
  - 신규 hooks 및 핸들러 추가 (handleRunJob, handlePauseJob, handleResumeJob)

### Test Updates
- `tests/services/news-crawler/test_crawler.py`: 이전 세션에서 수정한 레거시 테스트 변경사항 포함
